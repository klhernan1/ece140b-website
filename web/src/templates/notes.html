Design and Architecture

I use html and css for front-end design and styling. and I use python with pyramid for back-end. the addition of JS will be the pipieline from front end to back end for dynamic interactiona nd incase there was a need for existance of databse I will use MYSQL.


Currently I'm displaying the title in a hero element as shown in the video. I added a nav bar for easier navigation, specially after addition of other pages.
